## Points à vérifier lors de l'audit

### SEO On-Page
- [ ] Title tag unique et optimisé (<60 caractères)
- [ ] Meta description accrocheuse (150-160 caractères)
- [ ] H1 unique contenant le mot-clé principal
- [ ] Structure Hn logique (H2, H3...)
- [ ] Contenu unique et pertinent
- [ ] URL propre et descriptive

### SEO Technique
- [ ] Canonical URL correcte
- [ ] Schema.org LocalBusiness présent
- [ ] Temps de chargement (<3s)
- [ ] Mobile-friendly
- [ ] Core Web Vitals (LCP, FID, CLS)

### Maillage interne
- [ ] Liens vers services connexes
- [ ] Liens vers villes voisines
- [ ] Breadcrumb fonctionnel
- [ ] Lien vers page tarifs

### Conversion
- [ ] Numéro de téléphone visible
- [ ] CTA clair et visible
- [ ] Prix affiché
- [ ] Éléments de réassurance (avis, certifications)
